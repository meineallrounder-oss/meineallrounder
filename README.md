# Meine Allrounder - Website

Website für eine Bau- und Renovierungsfirma.

## Funktionen

- ✅ Responsive Design für alle Geräte (Desktop, Tablet, Mobile)
- ✅ Bildergalerie mit Filterfunktion
- ✅ Lightbox für Bilder mit Navigation
- ✅ Kontaktformular
- ✅ FAQ-Sektion mit Accordion
- ✅ Smooth Scrolling Navigation
- ✅ Mobile Menu

## Installation

1. Alle Dateien in einen Ordner kopieren
2. `index.html` im Browser öffnen
3. Keine zusätzlichen Installationen erforderlich!

## Bilder hinzufügen

Um eigene Bilder hinzuzufügen:

1. Erstellen Sie einen `images` Ordner im Hauptverzeichnis
2. Speichern Sie Ihre Bilder dort (z.B. `images/badsanierung-1.jpg`)
3. In `index.html` die Platzhalter-Bilder durch echte Bildpfade ersetzen
4. Beispiel: `<img src="images/badsanierung-1.jpg" alt="Badsanierung">`

## Anpassungen

### Kontaktdaten ändern
- Öffnen Sie `index.html`
- Suchen Sie nach Kontaktinformationen und passen Sie sie an

### Farben ändern
- Öffnen Sie `styles.css`
- Ändern Sie die CSS-Variablen im `:root` Bereich:
  ```css
  :root {
      --primary-color: #2563eb;
      --accent-color: #f59e0b;
      /* etc. */
  }
  ```

### Inhalte anpassen
- Alle Texte können direkt in `index.html` geändert werden
- Alle Stile können in `styles.css` angepasst werden

## Kontaktformular

Das Kontaktformular zeigt aktuell nur eine Bestätigungsmeldung. Um es funktionsfähig zu machen, können Sie:

1. Ein Backend-Script hinzufügen (PHP, Node.js, etc.)
2. Einen Service wie Formspree oder EmailJS verwenden
3. Die Formular-Daten an einen E-Mail-Service senden
